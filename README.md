### VSCode Setting for ESLint

- [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)

```json
"eslint.enable": true,
"eslint.validate": [
  "html",
  "javascript",
  "typescript",
  "javascriptreact",
  "typescriptreact"
],
"editor.codeActionsOnSave": {
  "source.fixAll": "explicit"
},
"[typescript]": {
  "editor.defaultFormatter": "dbaeumer.vscode-eslint"
},
"[javascript]": {
  "editor.defaultFormatter": "dbaeumer.vscode-eslint"
}
```
